Audio Waves — учебный Flask-проект для продажи битов.

Проект теперь разделен на блоки:
- app.py — основной файл запуска приложения;
- config.py — настройки проекта, пути к базе и папкам загрузки;
- extensions.py — подключение Flask-SQLAlchemy и Flask-Login;
- models.py — ORM-модели базы данных;
- services.py — вспомогательная логика: загрузка файлов, стартовые данные, внешний API;
- context_processors.py — общие данные для шаблонов, например счетчик корзины;
- routes/ — отдельные блоки маршрутов:
  - web.py — главная, каталог, страница бита;
  - auth.py — регистрация, вход, выход;
  - store.py — профиль, избранное, корзина, оформление;
  - admin.py — панель администратора;
  - api.py — REST API и внешний музыкальный API.

Что есть в проекте:
- главная страница с фирменным оформлением;
- каталог битов с фильтрами и сортировкой;
- цены в рублях;
- кнопки прослушивания превью;
- регистрация и авторизация;
- личный кабинет, избранное и корзина;
- админка для загрузки битов;
- Bootstrap 5 в интерфейсе;
- собственный REST API;
- интеграция со сторонним музыкальным API.

REST API проекта:
- /api/beats
- /api/beats/<id>
- /api/music/search?term=drake

Как запустить:
1. Открой терминал в папке проекта.
2. Установи зависимости:
   pip install -r requirements.txt
3. Запусти приложение:
   python app.py
4. Перейди в браузере по адресу:
   http://127.0.0.1:5000

Стартовый админ:
- логин: admin
- пароль: admin123

Примечание:
Музыкальный поиск работает через внешний iTunes Search API, поэтому для него нужен интернет.
